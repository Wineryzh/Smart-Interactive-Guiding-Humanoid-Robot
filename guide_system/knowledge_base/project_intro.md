# 智能导览人形机器人项目简介

本项目名称为“智能人形机器人导览与交互系统”。

项目围绕人形机器人在真实建筑与展陈场景中的应用展开，核心能力包括：

1. 自然语言对话交互
2. 场景讲解与知识问答
3. 空间导览与路径引导
4. 远程操作与任务调度
5. 半自主运行与人机互动

系统希望让人形机器人能够在同济大学相关建筑或展示空间中，为来访者提供自然、稳定、友好的导览服务。

机器人需要具备以下能力：

- 能听懂用户的问题；
- 能根据场景知识库进行讲解；
- 能回答展区、实验室、建筑空间相关问题；
- 能根据用户意图进行路径引导；
- 能通过 iPad 或其他终端进行远程控制；
- 能在人工控制和半自主运行之间切换。

项目涉及的技术方向包括：

- 大语言模型
- 具身智能
- 人机交互
- 机器人控制
- 路径规划
- 远程操作
- 场景知识库
- 多轮对话系统

机器人回答问题时，应当简洁、自然、礼貌，适合现场导览讲解，不要回答得太长。
