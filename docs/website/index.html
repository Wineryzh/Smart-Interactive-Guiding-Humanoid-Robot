01 / PROJECT BACKGROUND

让机器人理解空间，
也理解来访者

高校建筑包含实验室、展陈空间、公共服务区域等多种功能空间。访客不仅需要“到达目的地”，也需要自然、准确且具有情境感的讲解服务。

传统标识与固定终端难以提供连续的空间引导和动态问答。本项目以同济大学土木工程大楼室内环境为研究载体，探索人形机器人在复杂公共建筑中的感知、交互、移动与服务能力。

核心研究问题 如何构建可感知环境、可理解意图、可规划任务并能安全执行的智能导览闭环？

02 / RESEARCH OBJECTIVES

研究目标

面向真实室内环境，形成从访客需求理解到机器人服务执行的系统化研究框架。

O1

自然交互

理解访客问题与导览意图，支持自然语言问答和多轮交互。

O2

场景讲解

结合建筑空间知识，提供准确、简洁且适合现场表达的讲解。

O3

自主引导

研究室内定位、点位管理、路径规划及动态行走策略。

O4

人机协同

支持远程控制与半自主运行切换，保留必要的人工接管能力。

O5

安全运行

关注机器人状态监测、动作约束与真实场景下的系统稳定性。
