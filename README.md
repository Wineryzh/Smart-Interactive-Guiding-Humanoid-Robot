# Smart-Interactive-Guiding-Humanoid-Robot
Codebase for the Smart Interactive &amp; Guiding Humanoid Robot Research Project
## Project Structure

```text
Smart-Interactive-Guiding-Humanoid-Robot/
│
├── docs/              # Project documents, meeting notes, API design
├── dialogue/          # LLM-based dialogue and multi-turn interaction
├── guide_system/      # Tour guide scripts, scene knowledge base, explanation generation
├── navigation/        # Indoor/outdoor path planning and autonomous walking
├── teleoperation/     # Remote control and task scheduling system
├── robot_control/     # Humanoid robot SDK wrapper and low-level control interface
├── perception/        # Camera, object detection, person tracking, scene recognition
├── app_interface/     # iPad/Web control interface
├── config/            # Configuration files
├── data/              # Logs, maps, demo records, guide materials
├── scripts/           # Utility scripts
├── tests/             # Test scripts
└── assets/            # Images, diagrams, videos
